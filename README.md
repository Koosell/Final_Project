## 📋 Description

This is a final project built with **React** and **Vite**.  
It features:

- Routing with React Router  
- Modular components  
- Asset management (images, CSS)  
- Fast development and build using Vite  

---

## 🧰 Tech Stack

- ⚛️ React 18  
- 🚦 React Router DOM  
- ⚡ Vite  
- 🎨 CSS  
- 📦 Node.js & npm
