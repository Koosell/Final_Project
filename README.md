## 📋 Deskripsi

Ini adalah proyek final yang dibangun menggunakan **React** dan **Vite**.  
Proyek ini mencakup fitur:
- Routing menggunakan React Router
- Komponen modular
- Manajemen aset (gambar, CSS)
- Build cepat dengan Vite

---

## 🧰 Tech Stack

- ⚛️ React 18
- 🚦 React Router DOM
- ⚡ Vite
- 🎨 CSS
- 📦 Node.js & npm
